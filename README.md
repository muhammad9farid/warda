# Warda by Sara

An Arabic-first floral storefront for `Warda by Sara`, designed around handmade forever-flower gifting from Egypt.

## What is included

- `index.html` - storefront markup
- `styles.css` - visual design, layout, and responsive styling
- `app.js` - cart, language toggle, checkout flow, and custom bouquet interactions
- `assets/sq.png` - brand logo and favicon

## Run locally

Open `index.html` directly in a browser, or serve the folder with:

```bash
python -m http.server 4174 --bind 127.0.0.1
```

Then open:

```text
http://127.0.0.1:4174/
```
